## Description
Includes:
* [Koa](http://koajs.com/)
* [Jade](http://jade-lang.com/)
* [Stylus](http://learnboost.github.io/stylus/)
* [Gulp](http://gulpjs.com/)
* [jspm](http://jspm.io/)

## Installation and Usage
    
    $ git clone git@github.com:evenfrost/plate.esnext.git
    $ cd esnext
    $ npm install
    $ jspm install
    $ gulp
